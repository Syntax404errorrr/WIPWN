⚙️ Installation

pkg update && pkg upgrade -y
pkg install root-repo -y
pkg install git python wpa-supplicant pixiewps iw openssl -y
pkg install tsu -y || pkg install sudo -y
git clone https://github.com/anbuinfosec/wipwn
cd wipwn
chmod +x main.py

⚙️ Setup & Update

Install launcher script

python3 setup.py install

This will let you use the tool with:

wipwn --help

Uninstall launcher script

python3 setup.py uninstall

Update to latest version

python3 update.py

Make sure you're inside the wipwn directory when updating.

💡 Notes:
- Setup requires root access.
- Ensure git is installed for updates.
